Assignments, quizzes, and course notes for the **Django for Everybody Specialization** by University of Michigan on Coursera.

## About This Specialization

This specialization introduces Python programmers to building websites using the Django library. Across four courses, you will learn HTML, CSS, SQL, Django, JavaScript, jQuery, and JSON Web Services. During the course, you will build online websites using Django that are graded throughout the course.

This series is excellent preparation for learning other ways to build web applications using technologies like PHP (Web Applications for Everybody), Flask, or Ruby on Rails.

> Learners are encouraged to take the [Python for Everybody Specialization](https://www.coursera.org/specializations/python) or equivalent before beginning this specialization.

## Usage

Each course week folder contains the source code and notes for that week's assignments. To run any Django project:

```bash
cd <course-directory>/<week>/<project>/
python manage.py runserver
```

Make sure you have the required dependencies installed:

```bash
pip install -r requirements.txt
```
